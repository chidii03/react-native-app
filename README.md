# 🎬 CineNex | Premium Movie Discovery App

**CineNex** is a premium, high-performance mobile application built with **React Native** and **Expo**.  
It delivers a Netflix-inspired cinematic experience, powered by the **TMDB API** for real-time movie data and **Appwrite** for a scalable backend architecture.

---

## 🚀 Key Features

- **Premium UI/UX**  
  Sleek dark-mode interface designed with **NativeWind (Tailwind CSS)**, featuring smooth transitions and cinematic layouts.

- **Smart Search & Discovery**  
  Real-time search with trending and popular movie categories.

- **Netflix-Style Detail Pages**  
  In-depth movie details including dynamic backdrops, genre pills, budget & revenue stats, and production insights.

- **Appwrite Integration**  
  Secure backend for authentication, user data handling, trending algorithms, and global state synchronization.

- **Adaptive Layouts**  
  Fully responsive design optimized for all Android and iOS screen sizes.

---

## 🛠️ Tech Stack

### Frontend
- **Framework:** React Native (Expo SDK)
- **Routing:** Expo Router (File-based routing)
- **Styling:** NativeWind / Tailwind CSS
- **Language:** TypeScript
- **Icons:** Lucide React / Custom SVG Sets

### Backend & API
- **Backend-as-a-Service:** Appwrite (Authentication, Database, Cloud Functions)
- **Data Source:** TMDB API
- **Data Fetching:** Custom `useFetch` hook for optimized async operations

## 📦 Installation & Setup

### Clone the Repository
bash
git clone https://github.com/your-username/cinenex.git
cd cinenex
Install Dependencies
npm install
Environment Variables
Create a .env file in the root directory and add the following:

EXPO_PUBLIC_APPWRITE_ENDPOINT=https://cloud.appwrite.io/v1
EXPO_PUBLIC_APPWRITE_PROJECT_ID=your_project_id
EXPO_PUBLIC_TMDB_API_KEY=your_tmdb_api_key

# Start the Development Server
npx expo start
